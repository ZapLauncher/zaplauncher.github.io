Zap Launcher

A free, cross-platform, open-source Minecraft launcher using Eaglercraft (Electron) with a Python-based GUI.

Installation on Windows

Click on releases and download ZapLauncherInstaller.msi. Follow the installation steps. Once completed, you can choose to run it immediately or later using the checkbox.

⚠️ Warning (Windows): If you click No on the UAC prompt, Minecraft installation will fail due to denied access. So, don’t be silly — click Yes on the UAC prompt!

Installation on Linux

Click on releases, download ZapLauncherLinux.zip, and extract it. Open the extracted directory in your terminal using cd or by right-clicking and selecting “Open in Terminal”. You can use Konsole or your default terminal.

The executable name is zaplaunch.

⚠️ Warning (Linux): If you run bash install.sh and then run zaplaunch, the game may crash or show errors.

Why Zap Launcher is Better

  • ✅ It’s free and installs Minecraft without paying.
  • ✅ It’s cross-platform (Linux and Windows).
  • ✅ It’s open source — modify the source code freely.
  • ✅ Can be used in the command line.
